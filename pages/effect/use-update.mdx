# [useUpdate](https://ahooks.js.org/zh-CN/hooks/use-update)

> useUpdate 会返回一个函数，调用该函数会强制组件重新渲染。

## 部分源码

```tsx
const useUpdate = () => {
  const [, setState] = useState({});

  return useCallback(() => setState({}), []);
};

export default useUpdate;
```

在 React 中触发组件渲染主要有两种方式：
1. 通过 `setState` 更新状态，React 会自动触发组件重新渲染；
2. 父组件渲染时，子组件也会重新渲染；

所以，`useUpdate` 中通过设置一个空对象（每个空对象的引用都不一样）来触发 `setState`，
从而强制组件重新渲染，然后通过 `useCallback` 缓存这个函数并返回到外部使用。
