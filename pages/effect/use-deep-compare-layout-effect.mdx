# [useDeepCompareLayoutEffect](https://ahooks.js.org/zh-CN/hooks/use-deep-compare-layout-effect)

> 用法与 useLayoutEffect 一致，但 deps 通过 [react-fast-compare](https://www.npmjs.com/package/react-fast-compare) 进行深比较。

`useDeepCompareLayoutEffect` 的实现和 `useDeepCompareEffect` 类似，只是将 `useEffect` 替换为 `useLayoutEffect`。
具体可以参考 [`useDeepCompareEffect`](./use-deep-compare-effect) 的实现。
