# [useUpdateLayoutEffect](https://ahooks.js.org/zh-CN/hooks/use-update-layout-effect)

> `useUpdateLayoutEffect` 用法等同于 `useLayoutEffect`，但是会忽略首次执行，只在依赖更新时执行。

`useUpdateLayoutEffect` 的实现和 `useUpdateEffect` 类似，只是将 `useEffect` 替换为 `useLayoutEffect`。
具体可以参考 [`useUpdateEffect`](./use-update-effect) 的实现。
